## 우선순위 중간

### 2. 공대 생성 폼: 본인 캐릭터 입력 + 자동 연결
- [ ] 공대 생성 폼에 "내 캐릭터 이름", "내 캐릭터 직업" 필드 추가
- [ ] 공대 + 캐릭터 동시 생성 (isOwnerCharacter = true, uid = 생성자 uid)
- [ ] 기존 일반 공대원 등록 흐름과 분리 (uid = null 유지)
- 트리거: Step 4 완료 후 진행
- 영역: 공대 생성 모달 UI 확장
- 예상 작업량: 1~2시간
- 등록일: 2026-04-26
- 등록 사유: 공대 생성자 본인 캐릭터 자동 연결 UX (모델 C 채택)

## 우선순위 낮음 / 아이디어

### 5. 정산 숨김 기능 (통장정리)
- "내 정산 내역" 페이지에서 본인 시야에서만 숨김 (실제 데이터 삭제 아님)
- 데이터: `users/{uid}.hiddenSettlements = [settlementId, ...]`
- UI: 각 정산 행 [숨김] 버튼 + [숨긴 정산 보기] 토글 + [복원] 버튼
- 다른 공대원/공대장에게는 정상 표시, 본인만 안 보임
- 복구 가능
- 등록일: 2026-04-26
- 등록 사유: 정산 내역 개인 관리 (은행 통장정리 개념)
