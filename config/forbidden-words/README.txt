# 금칙어 관리 가이드

이 폴더는 공대명/파티명 생성 시 차단할 단어 목록을 관리합니다.

## 파일별 용도

- system.txt        : 운영진/관리자 사칭 방지
- scam.txt          : 사기/스팸 의심 단어
- inappropriate.txt : 테스트/장난 용어
- custom.txt        : 관리자가 자유롭게 추가하는 단어

## 편집 방법

1. 해당 카테고리 파일 열기
2. 한 줄에 단어 하나씩 추가
3. 파일 저장
4. 사이트 새로고침하면 즉시 반영

## 규칙

- 빈 줄은 무시됨
- # 으로 시작하는 줄은 주석 (무시됨)
- 대소문자 구분 없음 (admin 등록 → ADMIN도 차단)
- 부분 일치 (admin 등록 → AdminKing도 차단)

## 주의사항

⚠️ 너무 짧은 단어(2자 이하) 주의
   예: "bot" 등록 시 "로봇"도 차단될 수 있음

⚠️ 욕설 목록은 민감 정보
   필요시 .gitignore에 profanity.txt 추가

## 테스트

브라우저 콘솔에서:
  window.testForbiddenWord('테스트할단어')

결과가 true면 사용 가능, false면 차단됨.
